# Dean's Office Simulation

Simulaiton of Dean's Office written in Erlang.

## Getting Started

Inside erlang shell run this commands:
```
lc([deans_office, main, random_student, student_producer, students_queue, fifo]).
```
and then 
```
main:main().
```

### Prerequisites

Erlang VM.
